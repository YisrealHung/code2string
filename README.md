# code2string
製作這個是為了方便撰寫BlocklyDuino 積木程式時，可以把排整齊的Arduino cdoe 自動轉成String。同時也具備反轉的功能，對於參考別人的程式寫法很有幫助，也便於理解各功能的邏輯。
在同事的建議下，加入了自動複製至剪貼簿的功能按鈕。
